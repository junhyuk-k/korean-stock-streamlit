# 한국 주식 투자 지원 프로그램 제작 로드맵

## 프로그램의 최종 목적

이 프로그램은 자동매매 프로그램이 아니다.

국내 주식 전체 종목을 대상으로 차트, 공시, 국내 뉴스, 해외 뉴스,
정부 정책, 산업 흐름, 테마, 수급, 위험 신호를 종합 분석하여
상승 가능성이 높아지는 종목을 먼저 발견하고 그 이유와 위험을
사용자에게 설명하는 주식 투자 의사결정 지원 프로그램이다.

최종 매수와 매도 판단은 사용자가 직접 한다.

---

## 절대 원칙

- 자동매매 기능을 만들지 않는다.
- 계좌와 주문 기능을 연결하지 않는다.
- 한국 주식만 분석한다.
- 대형주와 중소형주를 모두 분석한다.
- 단순 급등 종목보다 상승 전 초기 신호를 우선한다.
- 추천 종목마다 상승 근거와 위험을 함께 표시한다.
- 공시와 뉴스를 중복 계산하지 않는다.
- 확인되지 않은 루머는 높은 점수를 주지 않는다.
- 추천 결과를 저장하고 실제 성과를 검증한다.
- 기능 제작을 완료한 후 영구 저장과 최종 배포를 진행한다.

## 작업 진행 규칙

- 한 번에 한 단계만 진행한다.
- 각 단계가 정상 작동한 것을 확인한 뒤 다음 단계로 이동한다.
- 코드 수정 전 반드시 백업한다.
- 코드 삽입 위치를 정확하게 확인한다.
- 들여쓰기 시작과 끝을 명확하게 확인한다.
- 오류가 발생하면 추측하지 않고 오류 문구를 확인한다.
- 시험 배포는 테스트용으로만 사용한다.
- 영구 저장과 최종 배포는 기능 제작이 끝난 뒤 진행한다.
